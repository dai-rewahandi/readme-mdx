# readme-mdx

Project ini pakai [mdx](https://mdxjs.com/) untuk membuat readme dengan [tailwindcss](https://tailwindcss.com/) dan [vite](https://vitejs.dev/).

## Getting Started

### Installation

```bash
git clone https://github.com/dai-rewahandi/readme-mdx.git

cd readme-mdx

bun install

bun run dev
```
